### 函数指针

​	使用`typedef`定义了一个新的函数指针类型

```
typedef void (*void_callback_uint32_ptr)(uint32 state, void *ptr);
```

​	之后直接使用这个新的类型定义一个数组

```
void_callback_uint32_ptr exti_callback_list[GPIO_GROUP_PIN_NUMBER_MAX] 
```
